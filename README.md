## 🗝️ Главные даты и факты для подготовки

### Конституция и права
- **1993 г.** — принята Конституция России (12 декабря)
- **10 декабря 1948 г.** — Всеобщая декларация прав человека (ООН)
- **20 ноября 1989 г.** — Конвенция о правах ребёнка (ООН)
- Ребёнком считается каждый **до 18 лет**
- «Конституция» = «устройство, постановление» (лат.)
- «Декларация» = «заявление» (лат.)
- «Федерация» = «союз, объединение» (лат.)
- Главная мысль Конституции: **«Человек, его права и свободы — высшая ценность»**

### Граждане России
- С **18 лет** — право голосовать на выборах
- Президент избирается на **6 лет**
- Президент — **Верховный главнокомандующий** Вооружёнными Силами
- **Федеральное Собрание** = Государственная Дума + Совет Федерации
- **Государственная Дума** принимает законы
- **Совет Федерации** одобряет или отклоняет законы
- Закон вступает в силу после подписи **Президента**

## 🎮 Правила игры

- **15 секунд** на каждый вопрос — следи за таймером!
- **4 варианта** ответа — выбери правильный
- За правильный ответ начисляются **очки + бонус за скорость**
- Серия правильных ответов даёт **множитель x1.5 → x2 → x3**
- **5 жизней** — не теряй их все!
- После каждого ответа — **интересный факт** из учебника
